# yolo-label

## 安装启动服务器

```bash
cd api && bun install && bun start
```

## 安装启动网页
```bash
cd ui && bun install bun run dev
```

This project was created using `bun init` in bun v1.3.12. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
